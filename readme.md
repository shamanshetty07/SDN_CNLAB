# 🌐 SDN-CNLAB

A collection of experiments and implementations for **Computer Networks Lab (CNLAB)** with a focus on **Software Defined Networking (SDN)** concepts.

---

## 📌 Overview
This repository contains lab programs, configurations, and notes related to:
- Computer Networks fundamentals
- SDN concepts using tools like Mininet / OpenFlow
- Networking protocols and simulations

---

## ✨ Features
- 🧪 Lab experiments with clear implementation
- 🌍 Networking simulations
- ⚙️ SDN-based setups and testing
- 📖 Easy-to-understand code and structure

---

## 🛠️ Tech Stack
- **Languages:** Python / C / Java (depending on experiments)
- **Tools:** Mininet, Wireshark, Cisco Packet Tracer
- **Concepts:** TCP/IP, Routing, Switching, SDN

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/shamanshetty07/SDN_CNLAB.git

# Navigate to project folder
cd SDN-CNLAB
```

### (Optional for SDN)
```bash
# Install Mininet (Ubuntu)
sudo apt update
sudo apt install mininet
```

---

## ▶️ Usage
- Navigate to the required experiment folder
- Run the program using appropriate commands (Python/C/Java)
- Analyze outputs and network behavior
- Use Wireshark for packet-level inspection (optional)

---

## 📚 Lab Experiments (Example)
- Experiment 1: Basic Networking Commands (Ping, Traceroute, Netstat)
- Experiment 2: Client-Server Socket Programming
- Experiment 3: Routing Algorithms (Dijkstra / Distance Vector)
- Experiment 4: SDN Topology using Mininet
- Experiment 5: Packet Analysis using Wireshark
